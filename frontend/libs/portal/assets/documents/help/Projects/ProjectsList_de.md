# <i class="mdi mdi-list-box-outline"></i> Projektverwaltung

## Überblick:

- Hier werden alle **Projekte** aufgelistet, denen Ihnen eine Rolle zugewiesen wurde
- Sie können Zeilen sortieren, indem Sie auf die Spaltenüberschriften klicken
- Sie können ein- und ausblenden, um Projektdetails ein- oder auszublenden
- Nur interne Benutzer können neue Projekte erstellen
