# Contributing Quick Start

Want to contribute to **Disuko**?  
Here’s how to get started quickly.

---

## 1. Sign the Eclipse Contributor Agreement (ECA)

Before your contribution can be merged, you must:

1. Create an Eclipse Foundation account
2. Sign the Eclipse Contributor Agreement (ECA)

➡ https://www.eclipse.org/legal/ECA.php

---

## 2. Find Something to Work On

- Check the issue tracker
- Look for issues labeled `good first issue` or `help wanted`
- Or propose an improvement by opening an issue

---

## 3. Submit Your Contribution

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Commit with a `Signed-off-by` line:  
`Signed-off-by: Your Name <your.email@example.com>`
5. Open a pull request

---

## 4. Review & Feedback

- A committer will review your pull request
- You may be asked to make changes
- Once approved, the PR will be merged

---

## 5. Communication & Conduct

Project discussions and decisions happen in public via:

- Issues and pull requests in this repository

By participating, you agree to the Eclipse Foundation Terms of Use:
https://www.eclipse.org/legal/termsofuse.php

All interactions follow the Eclipse Foundation Code of Conduct:
https://www.eclipse.org/org/documents/Community_Code_of_Conduct.php

---
