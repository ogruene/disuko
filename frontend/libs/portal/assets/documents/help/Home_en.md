# <i class="mdi mdi-view-dashboard theme--dark primary--text" style="font-size:16px;"></i> Dashboard

The FOSS Disclosure Portal **Dashboard** contains:

- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-ballot-outline" style="font-size:14px"></span> Your recent Projects
- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-file-check-outline" style="font-size:14px"></span> Your current Tasks
- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-shield-check-outline" style="font-size:14px"></span> A link to the FOSS License Database
- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-bank" style="font-size:14px;"></span> A link to the FOSS Policy Rules

## <i class="v-icon notranslate material-icons mdi mdi-menu"></i> Navigation:

- The side navigation provides direct access to the core functions of the system
- The breadcrumb on top provides a structure navigation
- The main content area provides further navigation elements
- The footer contains a link to the provider and privacy statement and to release notes
- Everything in blue performs an action
- Hyperlinks typically open in a new tab

## <i class="v-icon notranslate material-icons primary--text mdi mdi-help-circle-outline"></i> Context Help

When blue (active), this help action on the upper right corner provides you context information

## <i class="v-icon notranslate material-icons primary--text mdi mdi-account-circle-outline"></i> Profile:

The user profile on the upper rights contains different user settings and support links:

- **Profile:** Contains the metadata about your own user account
- **Support:** Open the support page
- **Appearance:** Switch between Dark Mode and Light Mode
- **Logout:** Log out of the FOSS Disclosure Portal
