# Lizenzentscheidungen

Einige FOSS-Komponenten werden unter verschiedenen Lizenzierungsoptionen angeboten. Das bedeutet, dass der **Projektverantwortliche** die Wahl treffen kann, welche Lizenz die beste ist. Es ist möglich, diese Wahl im FOSS Disclosure Portal zu dokumentieren, um zukünftige SBOM-Lieferungen auf der Grundlage dieser Entscheidung zu bewerten.

## Was muss ich tun?

- Einfach ausgedrückt: Die Entscheidung, welche Option verwendet wird, muss innerhalb des Projekts getroffen und dann im FOSS Disclosure Portal ordnungsgemäß dokumentiert werden.
- Bevor eine Lizenzwahl getroffen wird, wird in der Komponentenübersicht immer der Worst-Case-Status angezeigt. In der Spalte "Entscheidung" sehen Sie ein blaues "i"-Symbol, wenn es möglich ist, sich zwischen Lizenzen zu entscheiden.
- Wenn Sie vor einer Auswahl an Lizenzen stehen, verwenden Sie die freizügigste und bekannteste Lizenz. Wenn Sie wirklich eine Copyleft-Lizenz verwenden müssen, um ein Problem mit der Lizenzkompatibilität zu beheben, beachten Sie bitte die Konsequenzen und ergreifen Sie geeignete Maßnahmen, um diese zu mindern.
- Der "Projektverantwortliche" kann dann direkt innerhalb des Disclosure Portals dokumentieren, welche Lizenz ausgewählt wurde, so dass der angezeigte Policy-Status dies widerspiegelt.
- Info: Sie können den "Projektverantwortlichen" identifizieren, indem Sie in der Registerkarte Benutzer-Verwaltung des Projekts nachsehen. Dies ist die einzige Person im Projekt, der diese Funktion angeboten wird.

## Wo kann ich sehen, ob eine Entscheidung getroffen wurde, oder eine Entscheidung ändern?

- In der Projektübersicht sehen Sie im Reiter "Lizenzentscheidungen" alle Lizenzentscheidungen, die für dieses Projekt getroffen wurden.
- Um eine Entscheidung zu revidieren, kann der Projektverantwortliche die als Aktion angezeigte Abbruchfunktion nutzen. Die Entscheidung ist weiterhin in dieser Übersicht zu sehen, hat aber jetzt den gelben Status "Storniert".
- Anschließend können der Projektverantwortliche in der Komponentenübersicht eine neue Entscheidung treffen.
