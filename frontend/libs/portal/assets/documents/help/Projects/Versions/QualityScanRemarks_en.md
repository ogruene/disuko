# <i class="v-icon notranslate mdi mdi-text-search-variant primary--text"></i> Scan Remarks

- **Scope:** scan remarks provide a report on a set of data checks performed on the most current SBOM delivery. They do not provide a complete check of all attributes contained in the SBOM.
- **Warn Levels:**
  - <i class="v-icon notranslate material-icons mdi mdi-circle text-red" style="font-size: 12px;"></i> **Problem:** scan remarks on problem level (red) provide an indication that necessary information in the SBOM seems to be missing, incomplete, or unexpected and therefore cannot be processed properly; these problems must be investigated and resolved through the software supplier before the SBOM is approved.
  - <i class="v-icon notranslate material-icons mdi mdi-circle mdi mdi-circle text-orange" style="font-size: 12px;"></i> **Warning:** scan remarks on warning level (orange) provide an indication that further information in the SBOM seems to be missing, incomplete or remarkable; this indicator is also used in case a license declared and license concluded information differs; these warnings must be checked and if necessary be resolved through the software supplier.
  - <i class="v-icon notranslate material-icons mdi mdi-circle text-gray" style="font-size: 12px;"></i> **Information:** scan remarks on information level (grey) provide an indication that some information in the SBOM seems to be malformed; this indicator is also used in case a license alias has been applied.
