## Using unplugin, to import components
We use it to automatically import components.

## Arranging components sections & composition API
We use one file per component, and we use composition API. This is the best way to write components in
Vue 3.

Arrangement:

```vue
<script setup lang="ts"></script>
<template></template>
<style lang="scss" scoped></style>
```

## Syntactical structure for components
The development team was using a technical structure for components, but we do not want to use it anymore. Instead we 
use syntactical structure, which is more readable and easier to maintain and also better for ai.
