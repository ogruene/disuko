# Third Party Notice

## Overview:

- **Formats:** select the format most feasible for your software distribution. HTML, Plain Text and a simple JSON serialization are the formats available. The information contained in these formats is a reduction of the contents in the SBOM to the minimum set. The information contained in the different formats is the same, only the structuring differs.
- **Scope:** the third party notice displays _only open source and public domain licenses_ along with the related components, versions and copyright information. Notice information for proprietary, freeware software etc. (collectively "non-FOSS software") are not in the scope of the third party notice. FOSS Disclosure Portal users have to find their own method to fulfil such obligations from non-FOSS software and to get an approval for use of such licenses and software.
- **License Texts:** the license texts displayed are the template texts taken of the respective license entries from the license database.
- **Parameters:** the contact address displayed in the third party notice is a parameter which needs to be specified in the project settings under "contact address"; a valid contact address needs to be provided in external distributions.
