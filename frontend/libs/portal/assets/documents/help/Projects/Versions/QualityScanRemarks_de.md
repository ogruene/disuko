# <i class="v-icon notranslate mdi mdi-text-search-variant primary--text"></i> Prüfbemerkungen

- **Umfang:** Prüfbemerkungen enthalten einen Bericht über eine Reihe von Datenprüfungen, die für die aktuellste SBOM-Lieferung durchgeführt wurden. Sie bieten keine vollständige Überprüfung aller in der Stückliste enthaltenen Attribute.
- **Warnstufen:**
  - <i class="v-icon notranslate material-icons mdi mdi-circle text-red" style="font-size: 12px;"></i> **Problem:** Prüfbemerkungen auf Problemebene (rot) geben einen Hinweis darauf, dass notwendige Informationen in der SBOM zu fehlen, unvollständig oder unerwartet zu sein scheinen und daher nicht ordnungsgemäß verarbeitet werden können; Diese Probleme müssen durch den Softwarelieferanten untersucht und gelöst werden, bevor die SBOM genehmigt wird.
  - <i class="v-icon notranslate material-icons mdi mdi-circle text-orange" style="font-size: 12px;"></i> **Warning:** Prüfbemerkungen auf Warnstufe (orange) geben einen Hinweis darauf, dass weitere Informationen in der SBOM zu fehlen, unvollständig oder bemerkenswert zu sein scheinen; Dieser Indikator wird auch verwendet, wenn eine deklarierte Lizenz und eine abgeschlossene Lizenz unterschiedlich sind. Diese Warnungen müssen überprüft und gegebenenfalls durch den Softwarelieferanten behoben werden, bevor die Software verteilt wird.
  - <i class="v-icon notranslate material-icons mdi mdi-circle text-gray" style="font-size: 12px;"></i> **Information:** Prüfbemerkungen auf Informationsebene (grau) geben einen Hinweis darauf, dass einige Informationen in der SBOM fehlerhaft zu sein scheinen; Dieses Kennzeichen wird auch verwendet, wenn ein Lizenzalias angewendet wurde.
