# Token-Verwaltung

- **Geltungsbereich:** Sie können API Token für Ihre Projekte erstellen, damit Ihr Softwareanbieter die Disclosure Portal-API für ein bestimmtes Projekt verwenden kann. Wenn das Token auf Projektgruppenebene erstellt wird, ist es für alle untergeordneten Projekte unterhalb dieser Projektgruppe gültig. Wenn mehrere Lieferanten Daten für ein einzelnes Projekt bereitstellen, stellen Sie sicher, dass Sie für jeden von ihnen unterschiedliche Token erstellen, um die Rückverfolgbarkeit der SBOM Lieferungen zu gewährleisten.
- **Disclaimer:** Der Projekt-Token ist vertraulich zu behandeln und darf nur über verschlüsselte Kommunikationskanäle übermittelt werden. Übertragen Sie die Projekt-UUID aus Sicherheitsgründen nicht in derselben Nachricht wie das Projekttoken.
- **Gültigkeit:** Das Token hat eine definierte Lebensdauer, nach der es automatisch abläuft. Sie können ein aktives Token auch manuell ablaufen lassen. Folglich ist ein API Token so lange gültig, bis das Ablaufdatum erreicht ist oder bis es explizit widerrufen oder aktualisiert wird.
- **Übertragung:** Um ein API Token auf sichere Weise an Ihren Lieferanten zu übertragen, können Sie es als verschlüsselte E-Mail im Nachrichtentext senden.
- **Sicherheit:** Der Projekt Token ist vertraulich zu behandeln und darf nur über verschlüsselte Kommunikationskanäle übertragen werden. Übertragen Sie die Projekt UUID aus Sicherheitsgründen nicht in derselben Nachricht wie das Projekttoken.
- **Endpunkt:** Der zu verwendende Endpunkt hängt von der spezifischen Instanz des FOSS-Offenlegungsportals ab, auf der das Token generiert wird. Der Endpunkt wird nach dem Erstellen des Tokens angezeigt. In diesem Dialog haben Sie eine Aktion, um alle Informationen in Ihre Zwischenablage zu kopieren.
- **Automatisierung:** Um den Zugriff auf die FOSS Disclosure Portal API zu erleichtern, wird auf der Open Source Organization von eclipse-disuko auf öffentlichen GitHub.com ein [Command Line Interface Client (CLI)](https://github.com/eclipse-disuko/disuko-cli) bereitgestellt.
