# <i class="mdi mdi-view-dashboard theme--dark primary--text" style="font-size:16px;"></i> Dashboard

Das FOSS Disclosure Portal **Dashboard** enthält:

- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-ballot-outline" style="font-size:14px"></span> Ihre letzten Projekte
- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-file-check-outline" style="font-size:14px"></span> Ihre aktuellen Aufgaben
- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-shield-check-outline" style="font-size:14px"></span> Ein Link zur
  FOSS Lizenzdatenbank
- <span class="dashboard-card-icon material-symbols-outlined mdi mdi-bank" style="font-size:14px;"></span> Ein Link zu
  den Policy-Regeln

## <i class="v-icon notranslate material-icons mdi mdi-menu"></i> Navigation

- Die seitliche Navigation bietet direkten Zugriff auf die Kernfunktionen des Systems
- Die Navigationsleiste oben sorgt für eine strukturierte Navigation (Breadcrumb)
- Der Hauptinhaltsbereich bietet weitere Navigationselemente
- Die Fußzeile enthält einen Link zum Anbieter und zur Datenschutzerklärung sowie zu Versionshinweisen
- Alles in Blau führt eine Aktion aus
- Hyperlinks werden in der Regel in einem neuen Tab geöffnet

## <i class="v-icon notranslate material-icons primary--text mdi mdi-help-circle-outline"></i> Kontext Hilfe

Wenn diese Hilfeaktion in der oberen rechten Ecke blau (aktiv) ist, erhalten Sie kontextbezogene Informationen

## <i class="v-icon notranslate material-icons primary--text mdi mdi-account-circle-outline"></i> Profil:

Das Benutzerprofil oben rechts enthält verschiedene Benutzereinstellungen und Support-Links:

- **Profil:** Enthält die Metadaten zum eigenen Benutzerkonto
- **Support:** Öffnen Sie die Support Seite
- **Aussehen:** Umschalten zwischen Dunkelmodus und Hellmodus
- **Ausloggen:** Melden Sie sich vom Disclosure Portal ab
