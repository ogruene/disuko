## License Family

Licenses can be distinguished based on their license family. Permissive licenses impose only minimal restrictions on how the software can be used, modified, and redistributed, usually including a warranty disclaimer. Weak copyleft licenses only require that the source code of the original or modified work is made publicly available, and other code that is used together with the work does not necessarily inherit the original work’s license requirements. Strong copyleft requires that other code, that is used for adding, enhancing, and/or modifying the original work, also must inherit all the original work’s license requirements - such as to make the code publicly available. Network copyleft is usually strong copyleft, where the copyleft requirements are additionally triggered through the software being made available via a public network, such as the internet, for interaction by a user.
