# <i class="mdi mdi-list-box-outline"></i> Project Management

## Overview:

- Here relevant metadata for a project is shown
- Further metadata and settings can be accessed via "Edit" and "More"
- In the project tabs, you can manage different aspects of your project:
  - **Versions:** Manage the versions of the project for SBOM deliveries
  - **User Management:** Manage users of your project and edit their roles
  - **Token Management:** Manage tokens of your project for API access
  - **Policy Rules:** Display the allowed and denied licenses for the use case of the project

## Limits:

- Maximum versions per project: 10
- Maximum uploads per project per hour: 30
- Maximum file size per upload: 200 MB
