# Review-Bemerkungen

## Übersicht

- Review-Bemerkungen bieten einen Feedback-Kanal für Fragen oder Probleme mit den von einem Softwarelieferanten bereitgestellten Offenlegungsinformationen
- Solche Fragen oder Probleme werden als "Review-Bemerkungen" bezeichnet
- Zusätzlich zu einzelnen Review-Bemerkungen kann ein Gesamtbewertungsstatus verwaltet werden

## Lebenszyklus

- Review-Bemerkungen können aufgelistet, erstellt, kommentiert und in ihrem Status aktualisiert werden
- Nur der Ersteller einer Review-Bemerkungen kann die jeweilige Bemerkung bearbeiten, jeder andere Benutzer eines Projekts kann sie kommentieren
- Review-Bemerkungen beziehen sich auf Komponenten, Lizenzen und SBOM-Auslieferungen einer Projektversion und ändern sich daher im Laufe der Zeit in ihrem Status
- Nachdem eine neue SBOM geliefert wurde, die ein Problem behebt, kann die Review-Bemerkungen erneut überprüft und möglicherweise geschlossen oder in der Problemebene angepasst werden
- Review-Bemerkungen können nicht gelöscht werden; sie können in den Status "Geschlossen" oder "Abgebrochen" versetzt werden
