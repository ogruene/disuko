# <i class="mdi mdi-list-box-outline"></i> Projektverwaltung

## Überblick:

- Hier werden relevante Metadaten für ein Projekt angezeigt
- Weitere Metadaten und Einstellungen können über "Bearbeiten" und "Mehr" aufgerufen werden
- Auf den Projektregisterkarten können Sie verschiedene Aspekte Ihres Projekts verwalten:
- **Versionen:** Verwalten Sie die Versionen des Projekts für SBOM Lieferungen
- **Benutzer Verwaltung:** Verwalten Sie Benutzer Ihres Projekts und bearbeiten Sie ihre Rollen
- **Token Verwaltung:** Verwalten Sie Token Ihres Projekts für den API Zugriff
- **Policy-Regeln:** Zeigen Sie die zulässigen und verweigerten Lizenzen für den Anwendungsfall des Projekts an

## Grenzwerte:

- Maximale Versionen pro Projekt: 10
- Maximale Uploads pro Projekt und Stunde: 30
- Maximale Dateigröße pro Upload: 200 MB
