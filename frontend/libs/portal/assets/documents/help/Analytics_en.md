# Analytics

## Overview

- Search for used components and their licenses

## Data

- An initial import of the project can be triggered by an admin user using the /api/v1/analytics/export endpoint.
- The last SBom of a version is stored in the database and can be searched.
- Each time a new SBom is uploaded or a version is deleted the analytics repository is updated.
