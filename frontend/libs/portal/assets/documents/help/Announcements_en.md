# Announcements

## Overview

- This table lists the recent significant changes made to entries in the **FOSS License Database**
- Through this information related systems which implement rules regarding FOSS license entries can be kept in sync
- This feature and list is currently only available on the UI

## Dynamically Generated Content

- The entries displayed here are generated automatically in a job running nightly
- The information is derived from the audit log of the license entries
- Significant changes displayed here include:
  - License has received status "License Chart"
  - License has lost status "License Chart"
  - License has received the evaluation state "forbidden"
  - License which is in status "License Chart" has change on license familiy
  - License which is in status "License Chart" has change on license type
